# 前端工程化学习项目

## 项目目标

- 学习前端工程化基本流程
- 能够在实战中运用
- 学习后产出项目和文档

## 技术栈
- Vite
- Vue
- Ts

## 四周计划

### 第一周：搭骨架

- 创建项目，建立目标
- 代码规范接入
- 提交规范自动化
- 环境变量和配置管理
- 目录结构和路径别名
- 公共模块雏形
- 复盘

### 第二周：接近真实工程

- 路由和页面组织
- 组件分层
- 请求层完善
- 构建和脚本整理
- 样式和资源策略
- 补充文档
- 复盘

### 第三周：质量保障和自动化

- 测试环境接入
- 补 2 - 3 个基础测试
- CI 接入
- 类型和校验加强
- 打包分析和性能优化
- 错误处理和可维护性管理
- 复盘

### 第四周：部署，输出，总结

- 部署上线
- 发布流程最小化
- 代码与结构再整理
- 写技术总结
- 讲解演练
- 最终优化
- 总复盘方向
